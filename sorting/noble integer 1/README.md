Given N elements of unique numbers calculate number of noble integers?
A[i] is said to be Noble if
number of elements < A[i] = A[i]
<br>
<br>
<br>
Input : [ 1 -5 3 5 -10 4 ]
Output : 3
Explanation:
Array = [1 -5 3 5 -10 4]  
count = [2 1 3 5 0 4]
so 3 elements are Noble.

<br>
<br>
Input : [-3  0  2  5]
Output: 1
Explanation: 
Array : [-3  0  2  5]
count : [ 0  1  2  3]
so 1 element is noble
